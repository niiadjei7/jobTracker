JobTracker is a full-stack web application designed to help users track job applications, resumes, and application progress in a centralized dashboard. The project emphasizes clean API design, relational data modeling, and full-stack integration between a FastAPI backend and a React frontend.

End Goal Features

  User registration and login with secure password hashing
  
  Creation and management of job applications per user
  
  Tracking application status (Saved, Applied, Interview, Offer, Rejected)
  
  Notes and external links associated with each application
  
  User-specific application retrieval and display

Technical Highlights

  Backend to be built with FastAPI and PostgreSQL
  
  SQLAlchemy ORM for relational data modeling
  
  Pydantic schemas for request validation and response serialization
  
  Authentication logic implemented with bcrypt password hashing
  
  Frontend to be built with React, Axios, and Context API
  
  Backend and database containerized using Docker & Docker Compose
  
  RESTful API documented and tested via Swagger UI

Learning Objectives

  This project is being developed to strengthen understanding of:
  
  Backend API architecture and CRUD design
  
  Authentication workflows and secure credential handling
  
  Relational database modeling and query patterns
  
  Full-stack integration and client-server communication
  
  Containerized development environments
